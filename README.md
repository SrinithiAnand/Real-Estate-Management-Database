# SWEETHOME Real Estate Database System

## ✅ Overview
A complete database solution designed for the SWEETHOME real-estate platform.  
Includes conceptual design, relational model, SQL implementation, sample data population,
advanced queries, and NoSQL extension using MongoDB.

---

## ✅ Tech Stack
- Oracle SQL / PL/SQL
- MongoDB (NoSQL, JSON documents, Geospatial queries)
- ER/EER Modelling
- Object-Relational features in Oracle

---

## ✅ Features Implemented
✔ ER/EER conceptual design  
✔ Logical relational schema  
✔ SQL DDL to create 8+ normalized tables  
✔ Insert scripts with 50+ sample rows  
✔ Complex SQL queries using joins, filters, subqueries and date functions  
✔ Object-relational implementation using:
   - `OBJECT TYPE`
   - `REF` attributes
   - Object tables  
✔ NoSQL MongoDB version with:
   - Nested documents
   - Images & amenities arrays
   - Customer rental history
   - Geospatial indexing + near-location search

## ✅ How to Run
1. Run `DDL.sql` to create tables
2. Run `DML.sql` to insert data
3. Execute queries from `queries.sql`
4. MongoDB scripts can be run in Compass or mongosh
